This project provides STI College Alabang with a streamlined digital library management system that replaces manual processes with efficient QR code scanning and online book reservations. Designed for reliability, the user-friendly platform features a unique offline mode that saves data locally and syncs to the cloud once internet connectivity is restored, preventing workflow interruptions. By centralizing library tasks into one accessible web interface, the system enhances data security and provides a more organized, modern experience for both students and staff.
